# customer_behavior_analysis
End-to-end Data Analytics project analyzing customer behavior using Python, SQL, and Power BI. Includes data cleaning, exploratory data analysis (EDA), SQL queries, interactive dashboards, and business insights for data-driven decision-making.

# 📊 Data Analytics Project

## Overview

This project demonstrates an end-to-end Data Analytics workflow using industry-standard tools and technologies. It covers the complete analytics process, from loading and cleaning raw data to extracting insights with SQL, creating interactive Power BI dashboards, and presenting findings through a professional report and presentation.

The objective of this project is to showcase practical data analysis skills commonly used in real-world business environments.


## Dataset

The dataset contains business-related records used for analysis and visualization.

The project includes:

* Data loading using Python
* Data exploration (EDA)
* Data cleaning and preprocessing
* SQL-based data analysis
* Interactive dashboard creation
* Final report and presentation

## Project Steps

### 1. Data Loading

* Import dataset into Python.
* Verify data structure and columns.

### 2. Exploratory Data Analysis (EDA)

* Understand data distribution.
* Identify patterns and trends.
* Generate summary statistics.
* Create visualizations.

### 3. Data Cleaning

* Handle missing values.
* Remove duplicate records.
* Correct inconsistent data.
* Convert data types where required.

### 4. SQL Analysis

* Import cleaned data into:

  * PostgreSQL
  * MySQL
  * SQL Server
* Write SQL queries to answer business questions.
* Perform aggregations, joins, filtering, sorting, and grouping.

### 5. Dashboard Development

* Connect Power BI with the dataset.
* Build interactive dashboards.
* Add charts, KPIs, slicers, and filters.
* Present key business insights visually.

### 6. Reporting & Presentation

* Prepare a professional analytics report.
* Create a presentation using Gamma.
* Summarize findings and business recommendations.

## How to Run

1. Clone this repository.
2. Open the Python notebooks in Jupyter Notebook or VS Code.
3. Install the required Python libraries.
4. Run the notebooks in sequence:

   * Data Loading
   * EDA
   * Data Cleaning
5. Import the cleaned dataset into PostgreSQL, MySQL, or SQL Server.
6. Execute the SQL scripts.
7. Open the Power BI dashboard (`.pbix`) to explore visualizations.
8. Review the report and presentation for project insights.
